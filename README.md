# HpUp-Backend

## Database

All Migration and Models generate by sequelize-cli

### Run mirgation

`npx sequelize-cli db:migrate`

### Run seeds

`npx sequelize-cli db:seed --seed 20210124111220-usertype`
`npx sequelize-cli db:seed --seed 20210124111153-user`
